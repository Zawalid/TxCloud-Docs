## API Versioning

TXCloud uses date-based API versioning:

```
https://api.txcloud.io/v1
```

The current API version is `2025-01-01`.

### Version Header

You can pin a specific version:

```bash
curl https://api.txcloud.io/v1/... \
  -H "TXCloud-Version: 2025-01-01"
```
