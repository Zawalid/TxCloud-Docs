## Versioning de l'API

TXCloud utilise un versioning base sur la date :

```
https://api.txcloud.io/v1
```

La version actuelle de l'API est `2025-01-01`.

### En-tete de Version

Vous pouvez fixer une version specifique :

```bash
curl https://api.txcloud.io/v1/... \
  -H "TXCloud-Version: 2025-01-01"
```

## Politique de Depreciation

- Fonctionnalites depreciees avec 6 mois de preavis
- Changements majeurs annonces 12 mois a l'avance
- Notifications par email pour toutes les depreciations
