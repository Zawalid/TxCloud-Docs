## Rapports de Bugs

Vous avez trouve un bug ? Veuillez inclure :

1. Endpoint API et methode
2. Payload de la requete (masquez les donnees sensibles)
3. Reponse recue
4. Comportement attendu
5. ID de requete des en-tetes de reponse
