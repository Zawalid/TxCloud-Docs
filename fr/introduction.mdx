## Qu'est-ce que TXCloud ?

TXCloud est une plateforme API complete qui fournit des services de **verification d'identite**, **detection de fraude**, **surveillance des transactions** et **evaluation des risques de credit** adaptes aux marches emergents, avec une expertise approfondie en **MENA** et en **Afrique**.

## Apercu de la Plateforme

TXCloud fournit **7 familles d'API** qui fonctionnent ensemble de maniere transparente :

| API | Objectif | Fonctionnalites Cles |
|-----|----------|---------------------|
| **Identity** | Verification KYC | OCR de documents, correspondance faciale, vivacite |
| **FraudShield** | Detection de fraude | Intel appareil, velocite, listes de blocage |
| **TransactionGuard** | Securite des paiements | Scoring temps reel, analyse de patterns |
| **LendingRisk** | Evaluation de credit | Verification des revenus, accessibilite |
| **KYB** | Verification d'entreprise | Donnees entreprise, UBOs, dirigeants |
| **Watchlist** | Filtrage de conformite | Sanctions, PEP, medias negatifs |
| **DevEx** | Outils developpeur | Cles API, webhooks, analytics d'utilisation |

## Demarrage Rapide

Soyez operationnel en moins de 5 minutes :

<Steps>
  <Step title="Obtenez Votre Cle API">
    Inscrivez-vous sur [dashboard.txcloud.io](https://dashboard.txcloud.io) et creez votre premiere cle API.
  </Step>
  <Step title="Installez un SDK">
    Choisissez votre langage prefere et installez notre SDK officiel.
    ```bash
    npm install @txcloud/sdk
    ```
  </Step>
  <Step title="Faites Votre Premiere Requete">
    Verifiez votre premier document d'identite :
    ```javascript
    import TXCloud from '@txcloud/sdk';

    const txcloud = new TXCloud('votre_cle_api');

    const verification = await txcloud.identity.verify({
      document_front: 'image_base64...',
      country: 'MA',
      document_type: 'national_id'
    });

    console.log(verification.status); // 'verified'
    ```
  </Step>
</Steps>
