## Apercu

L'API KYB (Know Your Business) verifie les entites commerciales, recupere les donnees de l'entreprise, identifie les Beneficiaires Effectifs Ultimes (UBO) et filtre les dirigeants contre les listes de surveillance.

## Demarrage Rapide

```javascript
const verification = await txcloud.kyb.verify({
  company: {
    name: 'Acme Corp SARL',
    registration_number: 'RC12345',
    country: 'MA'
  },
  checks: ['registry', 'ownership', 'directors', 'screening']
});

console.log('Statut:', verification.status);
console.log('Raison Sociale:', verification.company.legal_name);
console.log('UBOs:', verification.ubos);
```

## Recherche d'Entreprise

Recherchez des entreprises avant la verification :

```javascript
const results = await txcloud.kyb.companies.search({
  query: 'Acme Corp',
  country: 'MA'
});

for (const company of results.data) {
  console.log(`${company.name} - ${company.registration_number}`);
}
```

## Structure de Propriete

Obtenez la chaine de propriete complete :

```javascript
const ownership = await txcloud.kyb.companies.getOwnership(companyId);

console.log('Structure de propriete:');
for (const owner of ownership.shareholders) {
  console.log(`${owner.name}: ${owner.percentage}%`);
  if (owner.type === 'company') {
    // Entreprise imbriquee - a ses propres actionnaires
    console.log('  Detenue par:', owner.shareholders);
  }
}
```

## Detection des UBO

Identifiez les Beneficiaires Effectifs Ultimes (25%+ de propriete) :

```javascript
const ubos = await txcloud.kyb.companies.getUBOs(companyId);

for (const ubo of ubos.data) {
  console.log(`UBO: ${ubo.name}`);
  console.log(`  Propriete: ${ubo.ownership_percentage}%`);
  console.log(`  Type de Controle: ${ubo.control_type}`);
  console.log(`  Verifie: ${ubo.verification_status}`);
}
```

## Filtrage des Dirigeants

Filtrez les dirigeants contre les listes de surveillance :

```javascript
const directors = await txcloud.kyb.companies.getDirectors(companyId);

for (const director of directors.data) {
  if (director.screening.pep) {
    console.log(`${director.name} est une PPE`);
  }
  if (director.screening.sanctions) {
    console.log(`${director.name} a une correspondance de sanctions`);
  }
}
```
